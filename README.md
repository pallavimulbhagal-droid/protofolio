## Setup

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173` in your browser.

### Build for Production

```bash
npm run build
```

## Customization

Edit the `constants.js` file to update:
- Your name, title, and bio
- Contact information (email, phone, location)
- Social media links
- Projects portfolio
- Work experience
- Skills
